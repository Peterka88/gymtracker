# GymTracker

A personal fitness tracking REST API built with Spring Boot. Track your workouts, monitor your lifts, log your nutrition, and follow your personal records over time.

## Features

- **Workout sessions** — log training sessions with date and notes
- **Exercise tracking** — manage exercises categorized by muscle group
- **Workout sets** — record sets with weight and reps for each exercise in a session
- **Personal records** — track your best lifts per exercise
- **Nutrition logging** — log daily calorie intake
